Fuel Tank Pie

Расширение добавляет в PILOT отдельную вкладку со списком машин, у которых обнаружены топливные датчики. При выборе машины показывается круговая диаграмма остатка топлива относительно полного бака.

Источники данных

Список машин и первичные значения датчиков загружаются из /api/v3/vehicles. При выборе машины текущий остаток уточняется через /api/v3/vehicles/instant-status.

Определение топливных датчиков

Расширение ищет датчики в sensors_status и fuel_sensors по именам, связанным с топливом: fuel, топл, ДУТ, бак. Служебные датчики батареи, GSM, типа топлива и расхода исключаются.

Полный бак

API V3 возвращает текущий остаток топлива, но не всегда возвращает емкость полного бака. Расширение сначала ищет емкость в данных машины, затем использует сохраненное в браузере значение. Если емкость не найдена, в поле "Полный бак" ставится оценка, которую пользователь должен уточнить.

Установка

Разместите содержимое папки fuel_tank_pie на публичном хостинге расширений и зарегистрируйте в PILOT расширение со slug fuel_tank_pie. Entry point: /Module.js.

Ограничения

Расширение не хранит данные на сервере. Значения полного бака, введенные пользователем, сохраняются только в localStorage браузера. Для диаграммы используется Highcharts, если он доступен в текущей сборке PILOT; иначе применяется CSS fallback.
